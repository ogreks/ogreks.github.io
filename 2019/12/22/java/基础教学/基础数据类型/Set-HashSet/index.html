Set 和 HashSet

Set 接口也是 Collection 接口的子接口, 它有一个很重要也是很常用的实现类—-HashSet.
Set 是元素无需并且不包含重复元素的Collection(List可以重复), 被称为集

HashSet 由哈希表(实际上是一个HashMap实例)支持. 它不保证set的迭代顺序; 特别是它不保证该顺序恒久不变
