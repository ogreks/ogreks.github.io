Hadoop 单机安装

Hadoop 启动模式

Hadoop 集群有三种启动模式：

  • 单机模式：默认情况下运行为一个单独机器上的独立 Java 进程，主要用于调试环境
  • 伪分布模式：在单个机器上模拟成分布式多节点环境，每一个 Hadoop 守护进程都作为一个独立的 Java 进程运行
  • 完全分布式模式：真实的生产环境，搭建在完全分布式的集群环境
